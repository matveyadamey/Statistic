
>[!def] **Определение**
>>[!Quote]   **Индикатор**
> — случайная величина с носителем $\{0,1\}$

>[!def] **Определение**
>>[!Quote]   **Распределение Бернулли**
> — распределение индикатора:
>> 
| k      | 0   | 1   |
| ------ | --- | --- |
| PMF(k) | 1-p | p   |


>[!def] **Определение**
>>[!Quote]   **Схема Бернулли**
> — конечная или бесконечная последовательность $\{X_k\}$ независимых и одинаково распределенных по Бернулли случайных величин


>[!def] **Определение**
>>[!Quote]   **Испытание Бернулли**
> — элемент схемы Бернулли

- Значение 1 в испытании называют успехом, 0 - неудачей

>[!seealso] **Лемма**
>>[!Quote]   **Свойства распределения Бернулли**
>>- $\mathbb{E}X=p$
>>- $\mathbb{E}(X^2)=p$
>>- $\mathbb{V}X=p-p^2=p(1-p)$
>

В Python:
```Python
import scipy.stats as sps
dist = sps.bernoulli(p)
```
